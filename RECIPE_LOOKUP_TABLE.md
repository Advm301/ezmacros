## Key Patterns

### 1. Pattern by Position in ID Range
Each protein type has an 8-recipe structure:
- Position 1-3: Base flavor variants (Soy Garlic, BBQ, Tomato)
- Position 4-6: Complex flavors (Spicy Gochujang, Pasta, Taco)
- Position 7-8: Additional variants (Stroganoff, Teriyaki)

### 2. Flavor Consistency Across Proteins
- All proteins have **identical flavor profiles**
- Only difference is protein source (beef vs chicken vs pork vs turkey)
- Macro values scale accordingly

### 3. EZ Level Pattern
- **EZ Level 1** (simpler): Positions 1, 3, 6, 8 in each protein range
  - Total: 16 recipes (4 per protein)
- **EZ Level 2** (moderate): Positions 2, 4, 5, 7 in each protein range
  - Total: 16 recipes (4 per protein)

### 4. Cook Method
- **ALL recipes use Skillet method**
- Reason: Fastest, most versatile, consistent 6-7 min active time

### 5. Macro Ranges by Protein

**Ground Beef Recipes:**
- Calories: 480-530 | Protein: 40-43g | Carbs: 50-54g | Fat: 10-14g

**Ground Chicken Recipes:**
- Calories: 450-500 | Protein: 42-46g | Carbs: 48-54g | Fat: 7-11g

**Ground Pork Recipes:**
- Calories: 490-540 | Protein: 37-40g | Carbs: 48-54g | Fat: 13-17g

**Ground Turkey Recipes:**
- Calories: 460-500 | Protein: 41-45g | Carbs: 48-54g | Fat: 8-12g

## Usage in App

### Example 1: User selects Ground Beef + Asian-Inspired flavor
```
pickRecipesByFlavorAndMethod('beef', ['Asian-Inspired'], 'Skillet')
Result: [92] → "Skillet Beef Soy Garlic Rice"
```

### Example 2: User selects Ground Chicken + Spicy flavor
```
pickRecipesByFlavorAndMethod('chicken', ['Spicy'], 'Skillet')
Result: [103] → "Spicy Gochujang Chicken Rice"
```

### Example 3: User selects Ground Turkey + Saucy flavor
```
pickRecipesByFlavorAndMethod('turkey', ['Saucy'], 'Skillet')
Result: [118, 122] → Two options:
  - 118: "Saucy Tomato Turkey Bowl"
  - 122: "Creamy Turkey Mushroom Skillet"
```

### Example 4: User selects Ground Pork + No flavor preference
```
pickRecipesByFlavorAndMethod('pork', [], 'Skillet')
Result: [108, 109, 110, 111, 112, 113, 114, 115] → All 8 pork recipes
```
